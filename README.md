## Prerequisites
* Node
* npm


## Build Setup

``` bash
# clone the project
git clone https://github.com/deepak-terse/react-boilerplate.git
cd react-boilerplate

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build
```

* react
